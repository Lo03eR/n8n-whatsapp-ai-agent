
# 🦷 AI-Администратор "Мерей" для стоматологии в Шымкенте

Интеллектуальный ассистент на базе **n8n** и **Google Gemini**, разработанный специально для автоматизации работы с клиентами через WhatsApp в стоматологической клинике "Аура" (Шымкент).

---

## 🚀 Описание решения

Данный воркфлоу позволяет клинике мгновенно реагировать на запросы клиентов 24/7, значительно снижая риск потери лидов из-за долгих ответов администраторов. Бот имитирует живого человека, понимает контекст и доводит клиента до записи на прием.

### ✨ Ключевые функции
- **Умный диалог:** Обработка естественного языка с помощью модели Gemini.
- **Двуязычность:** Понимает казахский (KZ) и русский (RU) языки.
- **Персонализация:** Запоминает имя клиента и использует его в общении.
- **Контекстная память:** Хранит историю диалога (до 10 последних сообщений), чтобы не переспрашивать информацию.
- **Целевое действие:** Фокус на записи клиента на конкретную услугу и время.

---

## 🛠 Стек технологий
- **[n8n](https://n8n.io/):** Платформа автоматизации (self-hosted).
- **[Google Gemini](https://ai.google.dev/):** Мозги ассистента для генерации ответов.
- **[Green-API](https://green-api.com/):** Интеграция с WhatsApp.
- **[Cloudflare Tunnel](https://www.cloudflare.com/products/tunnel/):** Доступ к локальному n8n без выделенного IP.

---

## ⚙️ Как это работает

1. **Входящее сообщение:** Клиент пишет в WhatsApp стоматологии.
2. **Webhook:** n8n принимает данные через Green-API.
3. **Фильтрация:** Система проверяет, что это именно текстовое сообщение.
4. **ИИ-обработка:** Агент Мерей анализирует вопрос, сверяется с памятью и формирует вежливый ответ.
5. **Отправка:** Готовый ответ уходит обратно клиенту через API.

---

## 📦 Установка

1. **Импорт:** Скачайте файл `shymkent_dental_ai.json` и импортируйте его в ваш n8n.
2. **Credentials:** - Настройте API-ключ для **Google Gemini**.
   - Настройте учетные данные для **Green-API** (Instance ID и Token).
3. **Конфигурация:** В ноде `HTTP Request` обновите URL, подставив ваши данные из Green-API.
4. **Запуск:** Настройте Cloudflare Tunnel и пропишите публичный URL в настройках Webhook в Green-API.
